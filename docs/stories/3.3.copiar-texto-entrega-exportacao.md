## Story
**As a** servidor da CGTE,
**I want** copiar o texto de cada entrega PGD individualmente com um clique,
**so that** eu cole diretamente no campo correspondente do Petrvs.

## Acceptance Criteria
1. Botao "Copiar" ao lado de cada entrega PGD que copia o texto para a area de transferencia
2. Feedback visual ao copiar (icone muda para âœ“ por 2 segundos)
3. Botao "Copiar Tudo" que copia o relatorio completo formatado
4. Opcao de exportar relatorio como arquivo texto (.txt) com todas as entregas
5. Texto copiado formatado de forma limpa (sem HTML, sem markdown â€” texto puro)
6. Funciona em desktop e mobile

## Dev Notes
- **Hook ja previsto:** `hooks/use-copy-feedback.ts` esta no source tree [Source: frontend-architecture.md#3]
- **Clipboard API:** `navigator.clipboard.writeText()` e assincrono e requer HTTPS (Vercel fornece por padrao)
  - Fallback para HTTP local: usar textarea temporario + `document.execCommand('copy')`
- **Texto Puro (AC: 5):** CRITICO â€” o Petrvs e um formulario web com campos de texto simples. O texto copiado NAO pode conter HTML tags, Markdown, ou caracteres de formatacao. Apenas texto UTF-8 limpo.
- **Download em Mobile:** `URL.createObjectURL` com `<a download>` funciona na maioria dos browsers mobile. Em Safari iOS antigo pode abrir em nova aba em vez de baixar â€” aceitavel para MVP.
- **Trimestre no nome do arquivo:** Converter "Mar-Mai/2026" para "mar-mai-2026" (lowercase, sem barra).
- **Depends on:** Story 3.1 (tela + pgd-entrega-list), Story 3.2 (motor de geracao de texto)
- **File Locations:** [Source: frontend-architecture.md#3]
  - `hooks/use-copy-feedback.ts`
  - `components/report/copy-button.tsx`
  - `components/report/copy-all-button.tsx`
  - `components/report/export-button.tsx`
  - `lib/report-formatter.ts`
  - Estender: `components/report/pgd-entrega-list.tsx`
- **shadcn/ui:** Usar `Button` com variante `outline` para botoes de acao
- **WCAG AA:** Touch targets >= 44x44px, aria-labels descritivos, feedback visual acessivel [Source: prd.md#3.4]
