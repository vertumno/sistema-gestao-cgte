## Dev Notes
- **Component Pattern:** O `KpiCard` ja tem exemplo na frontend-architecture [Source: frontend-architecture.md#4.1]
  ```typescript
  export function KpiCard({ label, value, trend, className }: KpiCardProps) {
    return (
      <section className={cn("rounded-lg border bg-card p-4", className)} aria-label={label}>
        <p className="text-sm text-muted-foreground">{label}</p>
        <p className="mt-1 text-2xl font-semibold tracking-tight">{value}</p>
        {trend ? <p className="mt-1 text-xs text-muted-foreground">{trend}</p> : null}
      </section>
    );
  }
  ```
  Estender para suportar `trendDirection` com cor condicional (verde/vermelho).
- **Entregas PGD Cobertas:** Usar `getTaxonomy()` da Story 1.3 para mapear categorias ativas -> entregas PGD unicas. Contar entregas que tem pelo menos 1 tarefa finalizada.
- **Variacao Periodo Anterior:** [Source: prd.md#Story 2.5 AC6]
  - Se periodo = trimestre atual (Mar-Mai/2026), periodo anterior = Dez-Fev/2026
  - Se periodo anterior nao tem dados, mostrar "N/A" ou "â€”" em vez de percentual

  - Mobile (< 1024px): `grid-cols-2` (2 colunas, ultimo card centralizado ou full-width)
- **Depends on:** Story 2.1 (layout + period selector), Story 2.2 (API /api/metrics + taxonomy integration)
