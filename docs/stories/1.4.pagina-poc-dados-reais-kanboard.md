## Story
**As a** coordenador da CGTE,  
**I want** ver uma pagina web que exibe dados reais do Kanboard (total de tarefas, tarefas por categoria),  
**so that** eu tenha confianca de que o sistema funciona antes de investir no dashboard completo.

## Acceptance Criteria
1. Route `/` exibe painel simples com dados reais do Kanboard
2. Mostra: total de tarefas, total por coluna (Backlog, Autorizado, Em andamento, Finalizado, Congelado)
3. Mostra: contagem de tarefas por categoria (as 18 categorias)
4. Dados carregados via Server Components (SSR) com revalidacao a cada 5 minutos
5. Estado de loading e tratamento de erro visiveis (se Kanboard indisponivel, mostra mensagem amigavel)
6. Layout basico com cores institucionais Cefor/Ifes (verde e branco) usando Tailwind
